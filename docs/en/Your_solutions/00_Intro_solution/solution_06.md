To find the local maxima or minima of the function $f(x) = 3x^2 - 12x + 7$, we can use the **First Derivative Test**. Since this is a quadratic function (a parabola), it will have exactly one vertex which represents either a minimum or a maximum.

---

### Step 1: Find the First Derivative

The derivative, $f'(x)$, represents the slope of the function. Local extrema occur where the slope is zero.


$$f'(x) = \frac{d}{dx}(3x^2 - 12x + 7)$$


Applying the power rule:


$$f'(x) = 6x - 12$$

### Step 2: Set the Derivative to Zero

To find the **critical point**, we solve for $x$ when $f'(x) = 0$:


$$6x - 12 = 0$$

$$6x = 12$$

$$x = 2$$

### Step 3: Determine if it is a Minimum or Maximum

We can use the **Second Derivative Test**. The second derivative $f''(x)$ tells us the concavity of the graph:

* If $f''(x) > 0$, the graph is concave up (a **minimum**).
* If $f''(x) < 0$, the graph is concave down (a **maximum**).

$$f''(x) = \frac{d}{dx}(6x - 12) = 6$$

Since $6 > 0$, the function is concave up at $x = 2$, meaning we have a **local minimum**.

### Step 4: Calculate the $y$-coordinate

Plug $x = 2$ back into the original function $f(x)$ to find the value of the minimum:


$$f(2) = 3(2)^2 - 12(2) + 7$$

$$f(2) = 3(4) - 24 + 7$$

$$f(2) = 12 - 24 + 7$$

$$f(2) = -5$$

---

### Final Result:

The function has a **local minimum** at the point **$(2, -5)$**. There are no local maxima for this function as it opens upward infinitely.
